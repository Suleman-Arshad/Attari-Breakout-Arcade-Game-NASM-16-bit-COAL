# Attari-Breakout-Arcade-Game-NASM-16-bit-COAL
Atari Breakout arcade game built in 16-bit NASM assembly as a COAL Semester 3 project. Features text-mode graphics via video memory, BIOS keyboard input, PC speaker sound effects, brick collision logic, scoring, lives, and win/lose screens. Developed by Muhammad Suleman &amp; Abdur Rehman.
